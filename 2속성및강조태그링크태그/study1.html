태그의 속성

태그의 속성은 시작태그에서 태그명 뒤 한칸띠어서 구분을 한다.

기본규칙- 속성 명 = "속성 값" (=는 =기준 우측의 값을 왼쪽에 저장한다 라는 의미로 사용됨)

링크태그

홈페이지를 다른페이지로 이동시켜줄 때 사용하는 태그

target이라는 속성을 사용해서 새 탭에서 페이지 이동을 시켜줄 수 있다. 기본값은 _self / 새탭에서 이동값 _blank

사용 시 줄바꿈이 되지않는다(인라인레벨태그)

네이버로 바로가기

절대경로와 상대경로

절대경로는 웹 상의 페이지 주소

상대경로는 현재파일 위치 기준 목적지 파일까지의 경로

. -> 현재파일이있는 폴더의 위치를 불러옴

/ -> 폴더를 연다

.. ->현재파일기준 상위폴더의 위치를 불러옴

welcome페이지로 바로가기 서브페이지1로 이동 서브페이지2로 이동

강조태그

강조태그란 특별하게 강조하고싶은 단어나 문장에 사용되는 태그

사용 시 줄바꿈이 되지않는다(인라인레벨태그)

strong,b,em,i,span
