입력양식태그

사용자로부터 데이터를 입력받기 위해 만드는 태그

form이라는 태그 공간안에 입력박스와 이름표를 구성할 수 있다.

입력양식 태그의 특징

  • 이름표태그(label)태그와 입력박스태그(input)태그는 사용자의 편의를 높이기 위해 id속성과 for속성의 값을 동일한 값으로 맞춰서 label을 클릭 했을 때 연결되어있는 입력박스태그가 활성화 되도록 한다.
  • 이 때 id값은 태그의 고유한 이름으로 사용되기 때문에 다른태그에서 중복적으로 사용할 수 없다.(class속성은 다른태그에서도 중복해서 사용할 수 있는 별명이다.)
  • input태그는 그 사용의 목적에 따라 type속성값을 다르게 적용하여 사용한다.
    • text - 간단한 한줄의 텍스트 입력
    • number - 숫자를 입력하게 할 때
    • password - 비밀번호를 입력하게 할 때
    • radio - 여러개의 기호 중 반드시 최소 한개를 선택하게 만들 때
    • checkbox - 자유롭게 선택하거나 취소하는 부분을 만들 때
    • submit - form태그안에 입력한 모든 데이터를 보내줄 때
    • reset - form태그안에 입력한 모든 데이터를 다지워서 초기화시킬 때

입력태그 안에서 사용되는 속성

  • placeholder - 문자를 입력했을 때 지워지는 가이드글 제시
  • disabled - 선택된 입력태그를 비활성화
  • selected - 미리 선택된 상태로 상태 수정
  • checked - 미리 체크된 상태로 상태 수정
  • name - 보내줄 데이터의 이름을 설정할 때 사용
  • value - name데이터에 저장되는 값을 설정할 때 사용
